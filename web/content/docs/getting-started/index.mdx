There are three ways to participate in the m1n3 protocol:

## Miner

Connect your SHA-256 mining hardware to a m1n3 stratum server. Your shares are submitted on-chain and you can trade them instantly on the marketplace.

[Miner Setup Guide →](/docs/getting-started/miner-setup)

## Node Runner

Run a Bitcoin full node and register blocks on the Sui blockchain. This is the **only way** to mint M1N3 tokens.

[Node Runner Setup Guide →](/docs/getting-started/node-runner)

## Pool Operator

Run the full m1n3 stack: Bitcoin node, gossip node, stratum server, and dashboard. Full control over your mining pool.

[Pool Operator Setup Guide →](/docs/getting-started/pool-operator)
